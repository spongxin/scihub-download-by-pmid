Create scihub_download/source_manager.py with:

1. **Class SourceManager** with:
   - `DEFAULT_SCI_HUB_SOURCES` = ["https://sci-hub.se", "https://sci-hub.ru", "https://sci-hub.red", "https://sci-hub.ee", "https://sci-hub.vg", "https://sci-hub.shop"]
   - `CACHE_FILE` = "~/.scihub_download/sources_cache.json"
   - `CACHE_TTL_HOURS` = 24

2. **scrape_sources()** method:
   - Fetch https://sci-hub.pub/ using requests with 10s timeout
   - Parse HTML with BeautifulSoup
   - Extract all sci-hub.* domain links from page
   - Return list of unique source URLs
   - Fallback to DEFAULT_SCI_HUB_SOURCES on any error

3. **test_source(source_url: str)** method:
   - Send HEAD request to source_url with 15s timeout
   - Return dict: {url, response_time_ms, status_code, available}
   - Handle exceptions gracefully, mark as unavailable

4. **test_all_sources(sources: List[str])** method:
   - Call test_source() for each source in parallel (max 5 workers)
   - Return list of SourceStatus dicts

5. **rank_sources(sources: List[SourceStatus])** method:
   - Sort by: success_rate DESC (primary), avg_response_time_ms ASC (secondary)
   - Return ranked list

6. **SourceStatus dataclass**:
   - url: str
   - success_count: int (default 0)
   - total_attempts: int (default 0)
   - avg_response_time_ms: float (default 999999.0)
   - last_checked: str (ISO timestamp)
   - available: bool (computed property: success_count > 0)

7. **Cache methods**:
   - `load_cache()`: Load from JSON, return dict of SourceStatus
   - `save_cache(sources)`: Save to JSON with pretty formatting
   - `is_cache_valid()`: Check if cache exists and < 24h old

8. **get_best_sources(n: int = 3)** main method:
   - Check cache validity
   - If cache valid: load, test top n, re-rank, return
   - If cache invalid: scrape, test all, rank, cache, return top n
